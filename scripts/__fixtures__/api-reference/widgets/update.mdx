---
title: 'Update widget'
api: 'PUT https://api.example.test/widgets/{id}'
description: 'Update an existing widget.'
---

<ParamField path="id" type="integer" required>
  Widget ID.
</ParamField>

<ParamField body="name" type="string">
  Widget name.
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT https://api.example.test/widgets/1 \
  -H "key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Renamed"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "name": "Renamed"
}
```
</ResponseExample>
