---
title: 'Get template'
api: 'GET https://api2.ecomailapp.cz/template/{id}'
description: 'Retrieve a specific template by its ID.'
---

<ParamField path="id" type="integer" required>
  Template ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api2.ecomailapp.cz/template/1234 \
  -H "key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```php PHP
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://api2.ecomailapp.cz/template/1234');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'key: YOUR_API_KEY',
    'Content-Type: application/json',
]);
$response = curl_exec($ch);
curl_close($ch);
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1234,
  "folder": 0,
  "name": "Some Template Name",
  "html": "<html>Html code here...</html>",
  "created_at": "2024-02-12 15:13:16",
  "updated_at": "2024-02-12 15:13:26"
}
```
</ResponseExample>
