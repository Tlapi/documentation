---
title: 'Show list detail'
api: 'GET https://api2.ecomailapp.cz/lists/{list_id}'
description: 'Retrieve detailed information about a specific list, including custom fields, groups, segments, and subscriber counts.'
---

<ParamField path="list_id" type="integer" required>
  ID of the list.
</ParamField>

<RequestExample>
```bash cURL
curl https://api2.ecomailapp.cz/lists/1 \
  -H "key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```php PHP
$listId = 1;
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api2.ecomailapp.cz/lists/{$listId}");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'key: YOUR_API_KEY',
    'Content-Type: application/json',
]);
$response = curl_exec($ch);
curl_close($ch);
```
</RequestExample>
