---
title: 'List all domains'
api: 'GET https://api2.ecomailapp.cz/domains'
description: 'Retrieve all domains in your account.'
---

<RequestExample>
```bash cURL
curl https://api2.ecomailapp.cz/domains \
  -H "key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```php PHP
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://api2.ecomailapp.cz/domains');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'key: YOUR_API_KEY',
    'Content-Type: application/json',
]);
$response = curl_exec($ch);
curl_close($ch);
```
</RequestExample>
