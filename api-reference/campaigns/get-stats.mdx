---
title: 'Get campaign stats'
api: 'GET https://api2.ecomailapp.cz/campaigns/{campaign_id}/stats'
description: 'Returns aggregated statistics for a specific campaign.'
---

<ParamField path="campaign_id" type="integer" required>
  Campaign ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api2.ecomailapp.cz/campaigns/1/stats \
  -H "key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```php PHP
$campaignId = 1;
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api2.ecomailapp.cz/campaigns/{$campaignId}/stats");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'key: YOUR_API_KEY',
    'Content-Type: application/json',
]);
$response = curl_exec($ch);
curl_close($ch);
```
</RequestExample>
